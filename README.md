# Music Video Project

A collection of tools for creating music videos from audio files using various scene selection and video generation strategies.

## Project Structure

- **Core Tools**: Current working generators and engines for music video creation
- **attic/**: Legacy code, test files, and old versions kept for reference
- **test-assets/**: Sample media files for testing (movie.mp4, song.m4a)
- **input/**: Directory for input files
- **memory-bank/**: Project documentation and context files

## Main Components

- `ultraRobustArchivalTool.py` - Primary archival video processing tool
- `premiere_style_archival_engine.py` - Premiere-style video editing engine
- `robust_music_video_generator.py` - Main music video generation engine
- Various specialized generators for different approaches

## Usage

Place your media files in the `input/` directory or use the sample files in `test-assets/` for testing.

See individual files for specific usage instructions.
